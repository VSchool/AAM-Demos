Three directions for the habit tracker.

The mobile-lane demo for Week 5 Day 4, built to the AAM demo guidelines. Pick one direction before any version code is written. Each is a complete product personality — not a palette variant of the other two.

How to read this

The thing to compare is not color — it's structure and personality. All three could be recolored toward each other and still be three different products, because they disagree about what a habit tracker is.

  • The status indicators are the most diagnostic artifact. Look at how each renders done / today / streak / skipped. A uses soft pills + a progress ring; B uses a tiny pill + tabular streak count; C uses inline typographic small-caps and a hairline — no badges at all. That single row tells you the whole direction.
  • Each direction renders in its own type and surface language below — Outfit on warm paper, Inter at Linear density, Fraunces on an editorial almanac page.
  • Swatches marked ★ are reserved status colors — each owns exactly one semantic role and appears nowhere decorative (guideline §7).
  • Every palette value is sampled from a real shipping product. The Sources section lists each one and is honest about which hexes were extracted from live CSS vs. inferred from teardowns (the iOS-only apps can't be inspected directly).

This is a mobile app — so where the guideline's web chrome calls for View Transitions and CSS bento, the build will substitute Reanimated equivalents (shared-element transitions, LayoutAnimation, spring entrances). The chosen codename here becomes the app's real name across all six versions.

Direction A

Hearth

codename candidate · "completion is a celebration"

A habit tracker should feel like a warm daily ritual, not a productivity ledger. Soft pastels, generous rounding, and a progress ring that fills as you go. Finishing a habit isn't a checkbox going gray — it's a small, satisfying moment with motion and color. The semantic palette is multi-hued on purpose: "done" green is never the only color, because the day is more than what you crossed off.

Anchored on Finch (emotional, mascot-driven completion · pastel multi-hue · heavy rounding) + Sunsama (warm orange accent · Outfit type · airy planning surface).

Direction B

Tally

codename candidate · "momentum you can measure"

A habit tracker is a precision instrument — fast, dense, and honest about the numbers. One sharp accent, monochrome everywhere else, tight rows you can scan in a glance. Streaks are tabular figures, not decorations; the point is the data, not the dopamine. This is the direction that feels like it belongs next to Linear and Things on a power user's home screen.

Anchored on Linear (monochrome + one indigo accent · Inter · dense rows · 1px borders over shadows · extracted from live CSS) + Notion Calendar (tabular numerals · tight display tracking · grid precision).

Direction C

Daybook

codename candidate · "a record you keep, not a game you play"

A habit tracker is a daily almanac — a page you turn, a record you keep. Editorial serif display, warm paper, generous margins, and status told through typography rather than badges: small-caps labels, a hairline rule, an inline streak count. The restraint is the point. It reads like a well-set journal, and it dares to be quiet where the other two are loud.

Anchored on Sunsama (warm editorial planning · Outfit/geometric · #F59549 warm accent) + Reflect (restrained two-color economy · whitespace-as-structure · content over chrome).
