W5D4 · Expo · Habit-tracker · UI-kit proposal

Notch — three directions for the same mobile habit tracker.

The C2 Week 5 Day 4 demo is an Expo habit tracker built across six progressive states (v0 through v5): scaffold, tabs, FlatList + form, swipe gesture, AsyncStorage, push notification. This proposal locks the visual language for Notch — the codename — before any v0 code is written. Tagline: "small marks, day after day."

The three directions below are structurally different, not palette variants. Each anchors on a distinct real reference (Things 3 + Sunsama; Linear; Reflect + HabitKit) and makes a different argument about what a habit tracker is. The most diagnostic artifact is the status indicator — three rows, three completely different shapes (filled square / bracketed flag / fourteen-cell weave). If those three shapes feel like the same idea, the proposal is wrong; if they feel like three different products, the proposal is doing its job.

How to read this. Don't compare colors. Compare structures. Direction A is a row with a swatch; B is a row with a bracketed flag; C is a row whose body is a 14-cell weave (the chart is the row). The typography differences are real (sans / mono-spec / serif) but secondary. Pick the direction whose status row best argues for the kind of mobile app you want students to ship — coach, console, or loom. Live --chrome inspection of all five reference web apps and six App Store listings is logged at the bottom of this file.
A · Quiet Companion

The phone is a journal, not a dashboard.

Warm paper canvas, one warm accent for "done today," small filled-square markers in the Things 3 tradition. Calm by default. Nothing competes with the habit titles.

Anchors: Things 3, Sunsama, Stoic
B · Console

A habit tracker that doesn't apologize.

Linear-grade dark surface, mono labels, bracketed status flags. Reads like a system log. For people who find pastels and rings patronizing. None of the six reference trackers do this — that's the point.

Anchors: Linear (mobile-equivalent), spec-sheet conventions
C · Loom

Each row is the chart.

Deep indigo canvas, per-habit accent threads, 14-day weave embedded in every habit row. The status indicator is the whole row body. Anchored on HabitKit's tile-grid pattern (4.9 / 2.1K ratings) plus Reflect's system polish.

Anchors: HabitKit, Reflect, Notion Calendar swatches
A · Quiet Companion

The phone is a journal, not a dashboard.

Direction A argues that a habit tracker should disappear into the day. Warm paper canvas borrowed from Sunsama, the small filled-square indicator from Things 3, and editorial restraint borrowed from Stoic's lowercase serif confidence. A single warm accent — Sunsama's signature orange #FF891C — is reserved for the only thing that matters: did you show up today. Streak counts live in a Things-3 blue as quiet metadata, not celebration. Missed days get a dashed brick outline that feels disappointed, not punitive.

B · CONSOLE

A habit tracker that doesn't apologize.

Direction B argues that the dominant aesthetic in this category — pastels, illustrations, ring animations, cheerful tab bars — gets in the way for people who already think in systems. Console borrows Linear's exact canvas (#08090A) and its restraint, runs all labels in Geist Mono, and replaces the cute checkbox with a bracketed flag: [ DONE ], [ STREAK 12 ], [ MISSED ]. Status reads like a system log. The violet is reserved for completion; amber for streaks ≥ 3; red for missed days; indigo for "scheduled today, not yet acted on." Nothing else gets a color. None of the six habit trackers surveyed do this — that's the differentiator.

C · Loom

Each row is the chart.

Direction C argues that you shouldn't have to tap into a detail screen to see how a habit is doing. Anchored on HabitKit (4.9 / 2.1K App Store ratings — the strongest validated tile-grid pattern in the category) and Reflect's deep-indigo system polish: the habit row's body is a 14-cell weave that fills as you keep showing up. Purple is the dominant accent; mint marks the peak day in a streak (#DDFCFA from Notion Calendar's swatch language); a soft pink signals a broken streak without making it punitive. Display copy in Fraunces gives the surface a journal-like, almost domestic feel — the opposite of B's spec-sheet, the opposite of A's quiet paper.

Sources sampled live

Reference sites and App Store listings inspected via --chrome.

Palettes, typography, and structural conventions below were extracted from live computed-style inspection in Chrome — not approximated from training data. Each web app was inspected with a sampler that walked the DOM and aggregated computed background-color, color, and font-family across the first ~2,000 elements, ranked by frequency. App Store listings were screenshotted with the phone-screenshot gallery in view.

Web app inspection (palette + typography tokens extracted live)

  • linear.app — canvas #08090A; surface #0F1011; elevated #23252A; indigo #6366F1; violet #8B5CF6; danger #EB5757; body font Inter Variable. Anchors Direction B's whole language.
  • culturedcode.com/things/ — cool-paper canvas #F2F5F7; surface #FFFFFF; ink #303336; brand blue #4F91FB; system sans (SF / ui-sans). Anchors Direction A's row composition and streak-meta color.
  • sunsama.com — warm paper #F6F5F1; signature orange #FF891C; DM Sans body. Anchors Direction A's canvas and reserved-accent choice.
  • reflect.app — deep-indigo canvas #030014; surface #060317; signature purple #6D48FF; Inter V body + AeonikPro display. Anchors Direction C's canvas and accent.
  • notion.com/product/calendar — soft swatch language #DDFCFA mint, #FFEBE3 peach, #F6F5F4 paper. Donated Direction C's peak-day mint.

App Store habit-tracker survey (structural conventions observed live)

  • Streaks — large circular gradient tiles in a 2×N grid; red-line stat charts. Validated that tile-grid is well-trodden; informed the choice to not compete on tiles.
  • Things 3 — small-icon list rows, filled-square checkbox, blue-accent restraint. Direct anchor for Direction A's row + marker pattern.
  • Habitify — white surface, blue pill segments, orange weekly heatmap on stats. Modern productivity flavor; informed Direction A's heatmap-as-detail-strip choice.
  • HabitKit — tile-grid charts where each habit is a horizontal swath of small colored cells (per-habit color, ~5×30 grid). 4.9 / 2.1K ratings — strongest validated structural analog for Direction C's "the row is the chart" argument.
  • Productive — dark teal canvas with friendly illustrations, central + FAB, day-strip header. Informed our decision against illustration-heavy "lifestyle" framing for a build-coach demo.
  • Strides — bright-blue brand, multi-color progress bars, calendar-day-circle stats. Classic charts-and-progress-bars; informed our move toward denser per-row state in Direction C.
  • Stoic — strict black-and-white palette, illustrated mark icons, lowercase serif display, editorial restraint. Reinforced Direction A's confidence to lean editorial instead of friendly-cartoony.

Sampled 2026-05-21. Palette and typography choices in each direction trace back to one or more of the sources above. The codename Notch, the tagline "small marks, day after day," and the structural arguments (row + swatch · row + bracketed flag · row-is-the-chart) are original to this proposal — informed by the survey, not derived from any single source.
