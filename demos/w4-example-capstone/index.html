✓ RUBRIC: Deployment Live on the internet via GitHub Pages

This site is deployed via GitHub Pages at vschool.github.io/AAM-Demos/demos/w4-example-capstone/. Deployment is automatic: push to the main branch, and GitHub Pages serves the updated files within minutes. No build step needed — the HTML, CSS, and JS files are served as-is.

For your capstone, you'll push your project to your own GitHub repo and enable Pages in Settings → Pages → select main branch. That's it — your project is live on the internet.

✓ RUBRIC: Value Solves a real problem with real data

Space Explorer solves a real problem: NASA publishes an incredible astronomy image every day, but their official page has no way to save favorites or browse the archive intuitively. This project adds that missing layer — a personal, searchable, saveable interface to 30 years of space imagery.

It uses a real public API (not mock data), handles real edge cases (videos vs. images, rate limits, missing dates), and provides real utility (favorites persist across sessions via localStorage).

Space Explorer — A W4 Capstone Example for V School's Applied AI Mastery program.
Data provided by NASA Open APIs.

Built with HTML, CSS, and vanilla JS only — no frameworks, no npm, no build tools.
